7.  情報の収集方法:

Cookieは小さなテキストファイルであり、様々な方法で我々を支援します。それによって、貴方様は我々のサイトにアクセスする時、より快適で有意義な体験をすることができます。たとえば、Cookie を使用すると、我々のWeb サイトにアクセスするたびにログインが不要になります。また、Webサイトや広告をカスタマイズして、貴方様の関心や好みに合わせてカスタマイズすることもできます。Cookieにはいくつかの異なるタイプがあります。

 

「永続性」Cookie は、指定された日付までコンピュータのハードドライブに保存されます。この日付は、明日、来週または 10 年後の日付にすることが可能です。永続性Cookieは、 a)が期限切れになるまで、b)更新された Cookie が上書きされるまで、または c)ユーザーが手動で削除するまで、コンピュータ上に保持されます。ほとんどのブラウザは Cookie を受け入れないように設定できますが、一部のサイトの機能や特性にアクセスできない場合があります。本サイトでは、貴方様が我々のオンラインストアから商品を購入する際の要件などを満足するには、サードパーティが提供する交付機能を使用する場合があります。この機能を実行すると、サードパーティがお使いのコンピュータに永続的なCookieを設定するか、ご購入の際に必要な個人情報を収集する可能性があります。

 

このWebサイトでは、サードパーティのポイント・アンド・クリック追跡分析ツール（DART、Omnitureラベル、DoubleClickとGoogle Analytics など）を使用してクリック統計を取得することができます。 www.auteltech.comは、 spotlight 広告/ Web ビーコン/ GIF/ピクセル・ラベルなどの Web 検出ツールを使用して、ページを閲覧したかどうかや閲覧回数などを判断することができます。我々より送信されたEメールまたは電子通信では、ピクセル・ラベルや Webビーコンなどのツールを使用してEメール・インディケータや情報を収集し、ユーザーの操作性を向上させることができます。たとえば、 Eメールメッセージの数、印刷や転送が行われているかどうか、使用しているデバイスの種類（モバイル・デバイスやコンピューターなど）、適用可能な IP アドレスに関連付けられている都市、州、国などです。通常、バナーなど、 Web ページの一部と見なされるすべての電子イメージは、 Web ビーコンとして機能します。

バナーまたは Web サイトへのリンクから、ユーザーはwww.auteltech.comに直接アクセスできます。www.auteltech.comによって掲載された広告または Web サイトへのリンクにはCookie またはWebビーコンを使用して、ユーザーの個人情報を収集しなく、広告またはリンクの有効性に関するメトリックを提供します。

 

　オンライン広告: Autel はサプライヤーと契約を締結し、 Autel 以外の Web サイトに広告を掲載することができます。貴方様はAutel以外のWebサイトにアクセスすると、Autel 広告を掲載するためにこれらのサプライヤーが Webトレース・ツール（Cookieや Web ビーコンなど）を使用してその前貴方様のオンライン活動情報を収集する可能性があります。さらに、 Autel 広告には、広告の有効性に関する情報を収集する Cookie や Web ビーコンなどの Web 追跡ツールが含まれている可能性があります。これにより、今後、対象となる広告やその他のメッセージをユーザーに送信します。これらのWebトレースツールは、 IP アドレス、アクセスページ、アクセス時刻などの情報を収集します。
