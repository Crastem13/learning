MaxiSys MS909

El MaxiSys MS909 se caracteriza por estar basado en una tableta con pantalla táctil de 10 pulgadas y ejecutarse sobre un sistema operativo Android 7.0. Está potenciado por un procesador Octa-Core (2.3GHz Quad + 1.7GHz Quad) y una memoria incorporada de 128GB, para proporcionar sus capacidades de Diagnóstico Guiado y Pruebas de Componentes para confirmar reparaciones. El MS909 dispone del nuevo MaxiFlash VCI, una combinación de interfase VCI para comunicación con el vehículo e interfase de reprogramación J2534 en uno, que le proporciona comunicaciones a red y al vehículo simultáneamente con su Wi-Fi dual más rápido. El MS909 cuenta con cámaras delantera y trasera y una batería de litio recargable que proporciona 8 horas de uso continuo, con el fin de garantizar que usted siempre tenga el acceso al diagnóstico.

  • 1. Actualizaciones Autel, que proporcionan la cobertura más reciente y a nivel OE para más de 80 marcas de vehículos Americanos, Asiáticos y Europeos, Incluyen vehículo de alta gama.

  • 2.  Funcionalidad excepcional a nivel OE, desde el OBDII, los Diagnósticos y Servicios específicos y hasta la Codificación y Programación avanzada de Computadoras ECUs.

  • 3.   Actualizaciones automáticas de Sistema Operativo y Software de Diagnóstico a través de internet, con mensajes de notificación en pantalla en tiempo real

  • 4.  Sesiones del Registro de Datos interactivos, que permiten tener contacto directo con el Soporte Técnico de Autel, para buscar solucionar a problemas y errores de diagnóstico.

  • 5.  Capacidad Multitarea Integral, diseñada para una gestión ideal de las operaciones del taller, para mantener bien organizados sus archivos de datos, información del cliente y registros del vehículo.

  • 6. El Data Manager en la nube le permite guardar sus registros de clientes y vehículos, datos de escaneo y notas técnicas agregadas.

  • 7. VCI con nuevos protocolos (D-PDU, RP1210, DoIP, CAN FD) para vehículos recientes



Sistema operativo

Android 7.0

Procesador

Procesador de ocho núcleos Samsung Exynos8895V (Mangosta de cuatro núcleos a 2.3 GHz + A53 de cuatro núcleos a 1.7 GHz)

Memoria

4 GB de RAM y 128 GB de memoria integrada

Pantalla

Táctil capacitiva TFT-LCD de 9.7 pulgadas, con resolución de 1536 x 2048 

Conectividad

  • WiFix2 (802.11 a / b / g / n / ac 2x2 MIMO)

  • BT v.2.1 + EDR

  • GPS

  • USB 2.0 (DOS USB HOST Tipo A, UN mini dispositivo USB)

  • HDMI 2.0

  • Tarjeta SD (admite hasta 256 GB)

CámaraTrasera


  • Posterior: 16 megapíxeles, enfoque automático, con flash

  • Frontal: 5.0 megapíxeles

Sensores

Acelerómetro de gravedad, sensor de luz ambiental (ALS)

Entrada / salida de audio

  • Micrófono

  • Altavoces duales

  • Conector de auriculares estéreo / estándar de 3 bandas de 3.5 mm

Energía y batería

  • Batería de polímero de litio de 15000 mAh y 3.8 V

  • Se carga mediante una fuente de alimentación de 12 V CA / CC

  • Con una temperatura entre 0 ° C y 45 ° C

Voltaje de entrada adaptador de 12V / 3ª
Temperatura de funcionamiento 

0 a 50 ° C (32 a 122 ° F)

Temperatura de almacenamiento -20 a 60 ° C (-4 a 140 ° F)
Dimensiones (WxHxD)

304.4 mm (11.98") x 227.8 mm (8.97") x 42.5 mm (1.67")

Peso

1.66kg (3.66 lb.)

Protocolos

DoIP, PLC J2497, ISO-15765, SAE-J1939, ISO-14229 UDS, SAE-J2411 Lata de un solo cable (GMLAN), ISO-11898-2, ISO-11898-3, SAE-J2819 (TP20), TP16, ISO -9141, ISO-14230, SAE-J2610 (Chrysler SCI), UART Echo Byte, SAE-J2809 (Honda Diag-H), SAE-J2740 (GM ALDL), SAE-J1567 (CCD BUS), Ford UBP, Nissan DDL UART con reloj, BMW DS2, BMW DS1, SAE J2819 (VAG KW81), KW82, SAE J1708, SAE-J1850 PWM (Ford SCP), SAE-J1850 VPW (GM Class2)
