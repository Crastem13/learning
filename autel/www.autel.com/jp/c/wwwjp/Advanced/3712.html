MaxiSys MS906BT

特定整備認証スキャンツール

MaxiSys MS906BTは、高度なワイヤレス診断デバイスのMaxiSys製品ラインに追加された最新製品です。このデバイスは、OEレベルの診断とECUコーディング機能の可能な限り最高のカバレッジを備えたAndroidオペレーティングシステムに基づいています。MaxiSys MS906BTは、強力なExynos 5260 6コアプロセッサ（1.3 GHzクアッドコアARM Cortex-A7 + 1.7 GHzデュアルコアARM Cortex-A15）、8インチ容量性HDタッチスクリーン、および長距離無線通信を効率的に利用します。包括的な車両診断と分析を提供します。

  • 仕様

  • 付属品

  • 1、非常に強力なサムスンエクシノスヘキサコアプロセッサ

  • 2、 8.0インチ1024x768 LEDバックライト付き光沢のあるマルチタッチ容量性ディスプレイ、IPSテクノロジーにより超移動性を実現

  • 3、パフォーマンスを向上させる内部の信頼できる64GB SSD

  • 4、オートフォーカスと懐中電灯を備えた内蔵8メガピクセルの背面カメラ

  • 5、お気に入りのウェブリソースにアクセスするための802.11b / g / n Wi-Fi

  • 6、ゴム製の外側保護と頑丈な内部ハウジングを備えたユニークな人間工学に基づいたデザイン

  • 7、内蔵の充電式リチウムポリマー電池により、最大14時間の連続使用が可能

  • 8、USB、オーディオ、補助ポートにより、アクセサリを簡単に接続できます

  • 9、高速起動とマルチタスクのためのオープンソースのアンドロイドオペレーティングシステム

  • 10、簡単で直感的なナビゲーションにより、必要なものを簡単かつ迅速に見つけることができます

  • 11、80を超える米国、アジア、ヨーロッパのメーカーとモデルに対応する最も広範な完全なシステムカバレッジ

  • 12、包括的なOBDII診断およびサービスから高度なECUコーディングまでの優れたOEレベルの機能

  • 13、コード、ライブデータ、アクティブテスト、ECU情報、適合、マッチング、コーディングなどの完全な機能

  • 14、車両を簡単に識別するスマートAutoVINテクノロジー

  • 15、インターネット経由のリアルタイムプッシュメッセージ通知によるシステムおよびソフトウェアの自動更新

  • 16、いつでもどこでもインスタントリモートコントロールの技術サポート

  • 17、クラウドベースのMaxiFixオンラインコミュニティは、診断および修復のヒントと実証済みの修正プログラムの膨大なデータベースを提供します

  • 18、対話型のデータロギングセッションにより、Autelテクニカルサポートに直接連絡して、診断のバグやエラーを直接トラブルシューティングできます。

  • 19、すべてのデータファイル、顧客情報、車両記録を適切に整理するためのワークショップ運営の理想的な管理のために設計されたワンストップマルチタスク

  • 20、ディスプレイタブレットとVCIデバイス間の強化されたワイヤレスBT接続により、機動性が向上し、作業が容易になります。

  • 21、内部プログラムアプリケーションを管理し、データを確認するための便利なデータ管理システム
